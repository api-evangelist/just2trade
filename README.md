# Just2TradeThis is a repo for managing machine readable artifacts for the [Just2Trade APIs](https://just2trade.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Just2Trade APIs](https://just2trade.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://just2trade.com)- [Developer](https://just2trade.com)- [Twitter](https://twitter.com/Just2TradeUSA)The [APIs.json](https://github.com/api-evangelist/just2trade/blob/master/apis.json) index provides links to all the relevant links for the Just2Trade APIs, including the OpenAPI and Postman if they are present.
